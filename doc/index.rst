ESPHome
-------

`ESPHome <https://esphome.io/>`_ is a system to control your ESP8266/ESP32 by simple yet 
powerful configuration files and control them remotely through Home Automation systems.

This section provides documentation for ESPHome related projects.

:doc:`zabbix`
   Integrate ESPHome devices with Zabbix network monitoring tool

:doc:`snmp`
    ESPHome component to integrate with network monitoring tools

:doc:`iec62056`
    ESPHome component to read data from electric and others utility meter.

:doc:`meter`
    How to build a device that can remotely read data from IEC 62056-21 compatible meter.
